# PerformanceLab

PerformanceLab is an open-source performance analysis toolkit for endurance athletes.

## Vision

Help athletes make better training decisions through physiological interpretation rather than simply displaying data.
Understand your training. Improve with confidence.

## Project Philosophy

PerformanceLab is guided by four core documents.

- MANIFESTO.md
- FOUNDATIONS.md
- ROADMAP.md
- SCIENTIFIC_PRINCIPLES.md

These documents define the long-term identity of the project and should be consulted before major architectural or scientific decisions.
